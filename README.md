# scalor

ANSI color

## usage

#### text color :

    "scalor-string" :# red

or

    "scalor-string".:#[Red]

#### background color :

    "scalor-string" :~ yellow

or

    "scalor-string".:~[yellow]

#### color tone :

    "scalaor-string" :# red :@ bright

or

    "scalaor-string".:#[red].:@[bright]

#### bold :

    "scalor-string" :@ bold

or

    "scalor-string".:@[bold]
